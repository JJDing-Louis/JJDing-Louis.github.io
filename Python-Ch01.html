1-1 函式輸出

Python的輸出函式為"print()"，結果會顯示於console上

                        
                        a = 123
                        b = 123.456
                        c='Python'

                        print(a)
                        print(b)
                        print(c)

                        print('a'=, a)
                        print('b'=, b)
                        print('c'=, c)

執行結果

                        
                        123
                        123.456
                        Python
                        a = 123
                        b = 123.456
                        c = Python

如果想不換行，可在print()裡面加上 end=' ' 

                        
                        a = 123
                        b = 123.456
                        c='Python'

                        print(a, end = ' ')
                        print(b, end = ' ')
                        print(c)

執行結果

                        
                        123 123.456 Python

end=' ' 裡面亦可加其他字元，作為分隔，例如:"/"

                    
                    a = 123
                    b = 123.456
                    c='Python'

                    print(a, end = '/')
                    print(b, end = '/')
                    print(c)

執行結果

                        
                        123/123.456/Python

1-1-2 %的格式化輸出

                    print(format(a,'5b'))

1-2 輸入函式

                    
                    >>> s =input()
                    Python
                    >>>print(s)
                    Python
                    >>> a = input('Enter a number: ')
                    Enter a number: 100
                    >>> print(a)
                    100

input()函釋回傳的結果是一個字串，而括號裡面可以放任何想在裡面顯式的訊號，例如:

                    
                    >>> a = input('Enter a number: ')
                    Enter a number: 100
                    >>> print(a)
                    100

"Enter a number:" 即為顯示的結果。

如果想將字串轉型為數字，則用int()或者eval()

                    
                    >>> a = int(input('Enter a number: '))
                    Enter a number: 100
                    >>> print(a)
                    100
                    >>>

若要一次輸入兩個資料時，則在輸入資料時，要加以逗號加以隔開。

                    
                    >>> a, b = eval(input('Enter two number: '))
                    Enter teo number: 66, 88
                    >>> print('a= %d, b = %d'%(a, b))
                    a = 66, b = 88
