0-1 Python 簡介

Python 是一種簡單上手且功能強大的直譯式的程式語言，本身是物件導向式的語言，其函式、模組、數字、字串都是物件。另外，Python支援繼承、重載、衍生、多重繼承的特性大大的增加了程式碼的複用性。由於寫法教完簡單易懂，可讀性佳以外，本身具有內建的函式庫做使用，而近期來許多社群的相互呼應與開放原始碼的風氣下，出現了不少大量的韓是褲可供使用。
Python的應用範圍很廣，可大致上如下:

  • 網路應用程式開發: Django、Pyramid、TurboGears、Tornado、web2py、Zope、Flask…
  • GUI設計: 本身包含的Tkinter庫能夠支援簡單的GUI開發，還可用PyGtk、PyQt與WxPython
  • 網路爬蟲: BeautifulSoup、Requests
  • 科學計算: Matplotlib、Pandas、NumPy

詳細的函式庫內容，可參考以下網址:

  • PyPI · The Python Package Index: https://pypi.org/
  • Python官方網站: https://www.python.org/

0-2 推薦的 IDE

市面上推薦的IDE成是繁多，不及備載，筆者目前先推薦用的順手的幾個IDE軟體:

PyCharm

由捷克公司JetBrains開發，提供代碼分析、圖形化調試器，集成測試器、集成版本控制系統，並支持使用Django進行網頁開發。有分成社群板與專業版，而社群版只提供Pytho的一般程式開發。

網站連結:PyCharm

VSCode

全名Visual Studio Code，是一個跨平台的程式編輯器且開放原始碼，內建了Git版本控制，同時內建了許多擴程式管理的功能，由Microsoft所開發。

網站連結:VS Code

Python語言的開放原始碼跨平台科學運算整合開發環境(IDE)整合了NumPy，SciPy，Matplotlib與IPython，以及其他開源軟體。本身整合於Anaconda的平台上，安狀Anaconda自東就會包含Spyder，也是目前證照考試常用的程式編輯器。

網站連結:Spyder
