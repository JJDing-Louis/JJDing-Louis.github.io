8-1 建立空字串

建立字串有兩種方式，一是以str()，二是以"表示之

                    
                        '''方法(一)'''
                        >>> s1 = str()
                        >>> s1
                        ''
                        '''方法(二)'''
                        >>> s2 = ''
                        >>> s2
                        ''

8-2 字串的運作

初始化字串有兩種方式，一是以str()，二是以"表示之

                    
                        '''方法(一)'''
                        >>> s3 = 'Learing Python now!'
                        >>> s3
                        'Learing Python now!'
                        '''方法(二)'''
                        >>> s4 = str('Learing Python now!'')
                        >>> s4
                        'Learing Python now!'

若要計暗算字串長度用len()，利用max()與利用min()函式分別計算字串的最大與最小值

                    
                        >>> len(s3)
                        20
                        >>>max(s3)
                        'y'
                        >>>min(s3)
                        ''

索引運算子 [] 用來擷取字串的某一字元

                    
                        >>> s3[3]
                        'r'
                        >>> s4 = 'Python'
                        >>> s4[-1]
                        'n'
                        >>> s4[-3]
                        'h'

使用分割運算子 [start: end] 表示擷取從start 到 end-1。

                    
                        >>> s4[1:4]
                        'yth'
                        >>> s3[:4]
                        'Pyth'
                        >>> s3[1:]
                        'ython'
                        >>> s3[1:-1]
                        'ytho'

+表示連接，而*表示複製

                    
                        >>> s5 = 'Bright'
                        >>> s6 = 'Tsai'
                        >>> s6 = ' Tsai'
                        >>> s5+s6
                        'Bright Tsai'
                        >>> s5*2
                        'BrightBright'

可用 in 或 not in 檢查字串是否包含該元素

                    
                        >>> 'B' in s5
                        True
                        >>> 'T' not in s6
                        False

                        >>> 'B' in s5
                        True
                        >>> 'T' not in s6
                        False

配合for敘述可以列印字串的所有元素值

                    
                        >>> for i in s5:
                            print(i, end=' ')

                        B r i g h t

8-3 測試字串

辭典(dictionary)由一鍵值(key)和數值(value)所組成的數對。

表6-2 有關串列運作之方法
方法 說明
isalnum() 若字串的字元是字母和數字所組成，則回傳True
isalpha 若字串的字元是字母所組成，則回傳True
isdigit() 若字串的字元是數字所組成，則回傳True
isdentifier() 若字串的字元是符合識別字元，則回傳True
islower 若字串的英文字元皆是由小寫字母所組成，則回傳True
isupper 若字串的英文字元皆是由大寫字母所組成，則回傳True
isspace 若字串的英文字元皆是由白色空白所組成，則回傳True

範例程式

                    
                        >>> s5 = 'Bright'
                        >>> s5.isalnum()
                        True
                        >>> s5.isalpha()
                        True
                        >>> s8 = 'Linda'
                        >>> s8.isalmun()
                        True
                        >>> s8.isalpha()
                        True
                        >>> s8.isdigit()
                        False
                        >>> s8.isidentifier()
                        True
                        >>> s8.isupper()
                        False
                        >>> s8.islower()
                        False
                        >>> s8.isspace()
                        False
                        >>> s9 = 'abcde'
                        >>> s9.islower()
                        True

8-4 子字串的運作

辭典(dictionary)由一鍵值(key)和數值(value)所組成的數對。

表8-4 子字串的運作方法
方法 說明
endswith(s1) 若字串的尾端是s1時，則回傳True
startswith 若字串的開頭是s1時，則回傳True
find(s1) 找尋字串中出現s1子字串的最小索引值，並加以回傳
rfind(s1) 找尋字串中出現s1子字串的最大索引值，並加以回傳
count(s1) 計算字串中出現s1子字串的個數

範例程式

                    
                        >>> s8 = 'Linda'
                        >>> s8.endswith('da')
                        True
                        >>> s8.startswith('Li')
                        True
                        >>> s8.find('d')
                        3
                        >>> s10 = 'abcdeabcde'
                        >>> s10.rfind('e')
                        9
                        >>> s10.count('e')
                        2

8-5 轉換字串

表8-3 轉換字串的方法
方法 說明
capitalize() 將字串中的第一個字轉成大寫，其餘的字轉換成小寫加以回傳。
lower() 將字串的所有字元轉換成小寫後加以回傳。
upper() 將字串的所有字元轉換成大寫後加以回傳。
title() 將字串中的每一個單字的第一個字轉成大寫，其餘的字轉換成小寫加以回傳。
swapcase() 將字串中的每一個單字的大寫轉小寫，小寫轉大寫後加以回傳。
replace(old, new) 將old字串以new字串取代之

範例程式

                    
                        >>> s11 = 'welcome to Taipei'
                        >>> s11.capitalize()
                        'welcome to taipei'
                        >>> s11.lower()
                        'welcome to taipei'
                        >>> s11.upper()
                        'WELCOME TO TAIPEI'
                        >>> s11.swapcase()
                        'WELCOME TO tAIPEI'
                        >>> s11.title()
                        'Welcome To Taipei'
                        >>> s11.replace('Taipei', 'Tainan')
                        'Welcome to Tainan'

8-6 如何從字串中去掉頭尾空白

表8-3 從字串中去掉頭尾空白的部分
方法 說明
lstrip() 將字串中的第一個字轉成大寫，其餘的字轉換成小寫加以回傳。
rstrip() 將字串的所有字元轉換成小寫後加以回傳。
strip() 將字串的所有字元轉換成大寫後加以回傳。

範例程式

                    
                        >>> s12 = '  Learing Python now!  '
                        >>> s12 
                        '  Learing Python now!  '
                        >>> s12.lstrip()
                        'Learing Python now!  '
                        >>> s12 
                        '  Learing Python now!  '
                        >>> s12.rstrip()
                        '  Learing Python now!'
                        >>> s12
                        '  Learing Python now!  '
                        >>> s12.strip()
                        'Learing Python now!'
                        >>> s12
                        '  Learing Python now!  '

8-7 如何將字串加以格式化

表8-3 將字串加以格式化的方法
方法 說明
center(width) 在給予 width的欄位寬下向中靠齊，並加以回傳。
ljust(width) 在給予 width的欄位寬下向左靠齊，並加以回傳。
rjust(width) 在給予 width的欄位寬下向右靠齊，並加以回傳。

範例程式

                    
                        '''置中靠齊'''
                        >>> s13 = 'Bright Tsai'
                        >>> s13.center(20)
                        '    Bright Tsai     ' 
                        '''向左靠齊'''
                        >>> s13
                        'Bright Tsai'
                        >>> s13.ljust(20)
                        'Bright Tsai         '
                        '''向右靠齊'''
                        >>> s13
                        'Bright Tsai'
                        >>> s13.rjust(20)
                        '         Bright Tsai'

split()方法，將字串解析到串列中，括號內的內容是用來判定分割的依據。

範例程式

                    
                        >>> s100 = 'Apple Orange Banana Kiwi'
                        >>> lst = s100.split()
                        >>> lst
                        ['Apple', 'Orange', 'Banana', 'Kiwi']
                        >>> s200 = '01-13-2018'
                        >>> lst2 =s200.split('-')
                        >>> lst2
                        ['01', '13', '2018']
