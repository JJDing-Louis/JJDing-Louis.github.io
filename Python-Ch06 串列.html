6-1-11 利用 for 印出串列所有的項目

                    
                        >>> lst3 = ['apple', 'orange', 'banana', 'kiwi']
                        >>> lst3
                        ['apple', 'orange', 'banana', 'kiwi']
                        >>> for i in range(len(lst3)):
                            print('lst3[%d] = %s'%(i, lst3[i]))

                        lst3[0] = apple
                        lst3[1] = orange
                        lst3[2] = banana
                        lst3[3] = kiwi

Python嘔提供一種新的方式，讓我們可以印出串列的所有項目，如下所示:

                    
                        >>> lst3
                        ['apple', 'orange', 'banana', 'apple']
                        >>> lst3[3] = 'kiwi'
                        >>> lst3
                        ['apple', 'orange', 'banana', 'kiwi']

若要連的項目所對應的位置也印出來，則可以用"for...in"迴圈表示之。

                    
                        >>> x = 0
                        >>> for i in lst3:
                                print('lst3[%d] = %s'%(x, i))
                                x+=1
                        lst3[0] = apple
                        lst3[1] = orange
                        lst3[2] = banana
                        lst3[3] = kiwi

表6-1 有關串列運作之函式
函式 意義
len 計算串列的長度
sum 加總串列每一個元素
max 回傳串列最大值
min 回傳串列最小值
表6-2 有關串列運作之方法
方法 意義
append(value) 附加value於串列的尾端
insert(indexp, value) 在索引indexp處加入 value
pop( 刪除串列最後一元素
pop(indexp) 刪除串列索引indexp的元素
remove(value) 刪除串列中的value，若有多個value，則只刪除第一個
count(value) 串列中出現value的個數
index(value) value所在串列的索引
sort() 將串列的元素由小到大排序
reverse() 將串列的元素反轉
表6-3 有關串列運作之運算子
方法 意義
in 檢視某一元素是否在串列中
not in 檢視某一元素是否不在串列中
[] 印出串列中的某一元素
[start: end] 印出串列從 start 到 end-1 的元素
* 複製多次的串列元素
+ 連接兩個串列元素

                        import random
                        lotto = []

6-2 二維串列的運作

二維串列的ˊ運作與一維串列的運作很像斯。以下範例說明之。

6-2-1 如何得知二維串列的列數與行數

範例程式:

                    
                        lst2 = [[1, 2, 3], [4, 5 ,6]]
                        print(lst2)
                        print(lst2[0])
                        print(len(lst2))
                        print(len(lst2[0]))

輸出結果:

                    
                        [[1, 2, 3], [4, 5 ,6]]
                        [1, 2, 3]
                        2
                        3

6-2-2 如何加入一維元素於二維串列

先記利空字串，再用append([])建立一串列，此時就建立二維串列了。

範例程式:

                    
                        import  random
                        rows = eval(input('Enter the number of row: '))
                        columns = eval(input('Enter the number of column: '))
                        
                        lst2 = []
                        for i in range(rows):
                            lst2.append([])
                            for j in range(columns):
                                lst2[i].append(random.randint(1, 50))
                        print(lst2)

輸出結果:

                    
                        Enter the number of row: 5
                        Enter the number of column: 3
                        [[37, 17, 18], [32, 20, 9], [10, 21, 21], [17, 47, 44], [18, 6, 22]]

6-2-3 列印二維串列的每一元素

利用for迴圈或者for...in迴圈，在指定串列的index即可印出所有內容，範例如下:

範例程式:

                    
                        import  random
                        rows = eval(input('Enter the number of row: '))
                        columns = eval(input('Enter the number of column: '))
                        
                        lst2 = []
                        for i in range(rows):
                            lst2.append([])
                            for j in range(columns):
                                lst2[i].append(random.randint(1, 50))
                        print()
                        for i in range(len(lst2)):
                            for j in range(len(lst2[i])):
                                print('lst[%d][%d} = %5d'%(i, j, lst2[i][j]))
                            print()

輸出結果:

                    
                        Enter the number of row: 5
                        Enter the number of column: 3
                        
                        lst[0][0] =    40
                        lst[0][1] =    33
                        lst[0][2] =    30
                        
                        lst[1][0] =    15
                        lst[1][1] =    50
                        lst[1][2] =    35
                        
                        lst[2][0] =     2
                        lst[2][1] =    43
                        lst[2][2] =    47
                        
                        lst[3][0] =    12
                        lst[3][1] =    15
                        lst[3][2] =     8
                        
                        lst[4][0] =     8
                        lst[4][1] =    27
                        lst[4][2] =    34
